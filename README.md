## 📖 Sobre el Proyecto

Portfolio profesional diseñado con una estética **city pop retro ochentera**, desarrollado para destacar mi experiencia en análisis de datos, proyectos y habilidades técnicas. Construido con tecnologías modernas para una experiencia fluida y responsiva.

### ✨ Características Principales

- 🎨 **Diseño City Pop**: Estética retro ochentera única y atractiva
- 📱 **Responsive Design**: Optimizado para todos los dispositivos
- ⚡ **Performance**: Construcción rápida con Vite
- 🎯 **Type-Safe**: Desarrollo con TypeScript
- 🎭 **Animaciones Suaves**: Transiciones y efectos visuales
- 📊 **Showcase de Proyectos**: Sección dedicada a proyectos de análisis de datos
- 💼 **Experiencia Profesional**: Timeline de mi trayectoria
- 🛠️ **Stack Tecnológico**: Visualización de habilidades y herramientas

## 🛠️ Instalación y Configuración

### Prerrequisitos

- Node.js (v16 o superior)
- npm o yarn

### Pasos de Instalación

1. **Clonar el repositorio**

```bash
git clone https://github.com/sguzman06/Sofia-Guzman-Portfolio.git
```

2. **Navegar al directorio**

```bash
cd Sofia-Guzman-Portfolio
```

3. **Instalar dependencias**

```bash
npm install
# o
yarn install
```

4. **Iniciar servidor de desarrollo**

```bash
npm run dev
# o
yarn dev
```

5. **Abrir en el navegador**

```
http://localhost:5173
```

## 🎨 Personalización

### Colores y Tema

Los colores del tema city pop pueden personalizarse en `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      // Tus colores personalizados aquí
    }
  }
}
```

## 🌐 Deployment

### Vercel (Recomendado)

1. Conecta tu repositorio de GitHub a Vercel
2. Vercel detectará automáticamente Vite
3. Deploy automático en cada push a `main`

### Otras Opciones

- **Netlify**: Drag & drop de la carpeta `dist`
- **GitHub Pages**: Configurar GitHub Actions
- **Railway**: Deploy directo desde GitHub
